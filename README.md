# Lupy Nails Agenda Pessoal Privada

- Projeto mobile-only para agenda pessoal privada da Lupy Nails.
- Deploy automático no GitHub Pages via GitHub Actions.
- Dados salvos no localStorage do dispositivo.
- Aplicação estática sem backend.

## Privacidade e segurança
Este projeto em GitHub Pages é estático e público. A senha local é apenas visual. Não inserir dados reais diretamente no código-fonte. Dados de uso ficam no localStorage do dispositivo.
